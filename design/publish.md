## Publish

Information for publishing

Reference: https://code.visualstudio.com/api/working-with-extensions/publishing-extension

Portal: https://marketplace.visualstudio.com/manage

```bash
# Install VSCE CLI

$ npm install -g @vscode/vsce

# Navigate to Project Root

$ cd typebox-script

$ vsce package     # Generate VSIX Package 
                   #
                   # Upload via Portal (or Authenticate via VSCE)
```
